# Homer

This project create a simple infrastructure in AWS.

### How to use  

```
$ git clone git@github.com:adrianocanofre/homer.git  
$ cd homer  
$ terraform init  
$ terraform plan
$ terraform apply  
```

### Features  

**IaC**  
  * New vpc
  * Subnet public and private
  * Nat Gateway  
  * Alb with target group
  * Security group for application
